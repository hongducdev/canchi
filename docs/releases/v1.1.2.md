# Can Chi v1.1.2

Round-launcher and splash polish.

## Highlights

- Round launcher icons use **full-bleed** calendar art (OS applies the circle mask — no tiny centered circle)
- Adaptive foreground keeps content in the safe zone
- Splash logo padded so Android 12+ circular splash does not over-zoom the calendar

## Install

1. Download `canchi-1.1.2.apk` from [GitHub Releases](https://github.com/hongducdev/canchi/releases)
2. Allow install from unknown sources if prompted
3. Open the APK and install
