# Can Chi v1.1.2

Round-launcher, splash, and adaptive-icon polish.

## Highlights

- Round launcher icons fill the circular mask (no tiny centered tile)
- Per-day **adaptive icons** for lunar-day launcher aliases (Android 8+)
- Splash logo padded so Android 12+ circular splash does not over-zoom
- Adaptive foreground sized at 108dp with content in the safe zone

## Install

1. Download `canchi-1.1.2.apk` from [GitHub Releases](https://github.com/hongducdev/canchi/releases)
2. Allow install from unknown sources if prompted
3. Open the APK and install

## Notes

- Uninstall the previous build (or clear launcher icon cache) if the old circular icon still appears after update
