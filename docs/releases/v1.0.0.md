# Lịch Âm v1.0.0

First public Android release of **Lịch Âm** — premium offline Vietnamese lunar calendar.

## Highlights

- Solar ↔ lunar calendar (Vietnam timezone)
- Can Chi, tiết khí, giờ Hoàng Đạo, day lore
- Festivals, Tết countdown, home-screen widgets
- Tools: lucky days, memorial calculator, feng shui, astronomy
- Văn khấn prayer catalog with profile fill
- Fully offline · no accounts · on-device storage

## Install

1. Download `licham-1.0.0.apk`
2. Allow install from unknown sources if prompted
3. Open the APK and install

> Signed with the project debug keystore for this first release (sideload / testing). For Play Store distribution, replace with a proper upload keystore.
