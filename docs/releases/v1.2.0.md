# Can Chi v1.2.0

Sao lưu Google Drive cá nhân (Android) và chỉnh trải nghiệm tab / Cài đặt.

## Highlights

- Sao lưu / khôi phục lên Google Drive (`Can Chi/canchi-backup.json`) — đăng nhập Google trong app
- Backup JSON v2 gồm hồ sơ + gia đình (xuất local và Drive dùng chung)
- Khối Cài đặt Drive gọn: Kết nối → Sao lưu / Khôi phục
- Tab bar tôn trọng thanh điều hướng hệ thống (không đè lên)

## Install

1. Download `canchi-1.2.0.apk` from [GitHub Releases](https://github.com/hongducdev/canchi/releases)
2. Allow install from unknown sources if prompted
3. Open the APK and install

## Notes

- Google Drive chỉ có trên bản Android (APK / development build), không dùng Expo Go
- Cần cấu hình OAuth (Web Client ID trong `.env`) phía nhà phát hành trước khi build
