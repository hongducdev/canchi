# Can Chi v1.3.0

La bàn và Thước Lỗ Ban trong tab Tiện ích.

## Highlights

- **La bàn** — hướng realtime (magnetometer), độ và 24 sơn; chỉ native
- **Thước Lỗ Ban** — kéo đồng bộ 3 thước 52,2 / 42,9 / 38,8 cm
- Cung lớn + cung nhỏ, đỏ dùng / đen bỏ, nhập mm hoặc cm
- Tra kết quả cả 3 thước cùng lúc + gợi ý kích thước tốt gần nhất

## Install

1. Download `canchi-1.3.0.apk` from [GitHub Releases](https://github.com/hongducdev/canchi/releases)
2. Allow install from unknown sources if prompted
3. Open the APK and install

## Notes

- La bàn cần bản native (không dùng trên web)
- Bản này vẫn ký bằng debug keystore (nội bộ / GitHub Releases)
