# Can Chi v1.1.1

Polish release for launcher icon and About update CTA.

## Highlights

- Center lunar day number in the white body of the calendar app icon
- Stronger **Kiểm tra cập nhật** button in Settings → Về ứng dụng

## Install

1. Download `canchi-1.1.1.apk` from [GitHub Releases](https://github.com/hongducdev/canchi/releases)
2. Allow install from unknown sources if prompted
3. Open the APK and install
