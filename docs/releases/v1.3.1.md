# Can Chi v1.3.1

Logo ngày âm tự cập nhật mỗi ngày; sửa widget bị cắt / lệch trên máy OEM (Honor).

## Highlights

- **Logo launcher** — đồng bộ ngày âm lúc ~00:05, sau boot / đổi ngày / múi giờ (không cần mở app)
- **Widget** — lấp đầy khung trên launcher báo sai kích thước (`fitXY`)
- Lưới tháng 2×2 / 4×2: khoảng cách, căn lề, chữ ngày âm rõ hơn; font không phóng theo cỡ chữ hệ thống

## Install

1. Download `canchi-1.3.1.apk` from [GitHub Releases](https://github.com/hongducdev/canchi/releases)
2. Allow install from unknown sources if prompted
3. Open the APK and install

## Notes

- Sau khi cài: mở app một lần để kích hoạt lịch logo; xóa rồi thêm lại widget để nạp layout mới
- Bản này vẫn ký bằng debug keystore (nội bộ / GitHub Releases)
