# Can Chi v1.1.0

Rebrand and polish release of **Can Chi** — Vietnamese lunar calendar.

## Highlights

- Official product name **Can Chi** across app, launcher, widgets, and docs
- Calendar-style app icon; on Android the launcher icon shows today’s **lunar day**
- About: author **hongducdev · hongduc.dev** + check for updates from GitHub Releases
- Auto update prompt (at most once per day) when a newer release is published

## Install

1. Download `canchi-1.1.0.apk` from [GitHub Releases](https://github.com/hongducdev/canchi/releases)
2. Allow install from unknown sources if prompted
3. Open the APK and install

## Notes

- `applicationId` remains `com.canchi.app` (update over previous Can Chi / Lịch Âm package builds)
- Deep links still use `licham://` this release for widget compatibility
- Data stays on-device; update check only calls the public GitHub Releases API
