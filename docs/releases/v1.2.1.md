# Can Chi v1.2.1

Cài cập nhật trong app — không cần mở trình duyệt.

## Highlights

- **Tải bản mới** tải APK về máy và mở trình cài đặt hệ thống
- Quyền `REQUEST_INSTALL_PACKAGES`; nếu chưa bật nguồn cài, app mở đúng trang cài đặt
- Giữ kiểm tra cập nhật tự động (≤1 lần/ngày) và nút trong Cài đặt

## Install

1. Download `canchi-1.2.1.apk` from [GitHub Releases](https://github.com/hongducdev/canchi/releases)
2. Allow install from unknown sources if prompted
3. Open the APK and install

## Notes

- Lần đầu cập nhật trong app có thể cần bật “Cho phép từ nguồn này” cho Can Chi
- Bản này vẫn ký bằng debug keystore (nội bộ / GitHub Releases)
