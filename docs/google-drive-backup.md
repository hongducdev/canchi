# Sao lưu Google Drive

Trên Android, vào **Cài đặt → Dữ liệu → Sao lưu Google Drive** để kết nối tài khoản Google và sao lưu / khôi phục dữ liệu trên máy bạn.

- File trên Drive: `Can Chi/canchi-backup.json`
- Dữ liệu chỉ rời máy khi bạn bấm **Sao lưu ngay**
- Can Chi không có máy chủ đồng bộ riêng

Khôi phục sẽ thay thế dữ liệu trên máy theo bản sao lưu (có xác nhận trước).
